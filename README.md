# loopup-chat

A modern chat application designed for seamless communication and collaboration.

## Features

- **Real-time Messaging**: Instant text-based communication.
- **Cross-Platform Support**: Accessible on web, desktop, and mobile devices.
- **User-Friendly Interface**: Clean and intuitive design for easy navigation.
- **Customizable Themes**: Personalize the look and feel of your chat experience.
- **Secure**: End-to-end encryption for enhanced privacy.

## Installation

To set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/TranTrungHiu/loopup-chat.git
   ```
2. Navigate to the project directory:
   ```bash
   cd loopup-chat
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage

1. Open the app in your browser at `http://localhost:3000`.
2. Register or log in to start chatting with your friends or team.
